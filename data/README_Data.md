# Data 

This folder contains a sample version of the cleaned retail dataset used in the project.

The full cleaned dataset was not uploaded because it exceeds GitHub's file size upload limit.

The project was built using the Online Retail II dataset, which includes invoice-level retail transactions with fields such as:

- Invoice
- StockCode
- Description
- Quantity
- InvoiceDate
- Price
- Customer ID
- Country
- Revenue
- Category

The complete dataset was cleaned in Python and used for both the Marketing Analytics and Supply Chain Analytics notebooks.
